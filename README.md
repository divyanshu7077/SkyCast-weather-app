# 🌤️ SkyCast Weather App

A modern weather application built using React, Vite, and TanStack Query.  
It provides real-time weather data, forecasts, and a clean UI experience.

---

## 🚀 Features

- 🌍 Search weather by city
- 📍 Current location weather
- 📊 Hourly temperature graph
- 📅 5-day forecast
- 🌙 Light/Dark mode
- ⚡ Fast data fetching with TanStack Query

---

## 🛠️ Tech Stack

- React + Vite
- TypeScript
- TanStack Query
- Tailwind CSS
- Recharts
- OpenWeather API

---

## ⚙️ Setup

1. Clone the repo:
```bash
git clone https://github.com/divyanshu7077/SkyCast-weather-app.git
cd SkyCast-weather-app

2. install dependencies:
``` bash
npm install

3.Create a .env file:

  VITE_WEATHER_API_KEY=your_api_key_here

4.Run the app:
```bash
npm run dev
